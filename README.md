# minesweep
- A game about clearing the minefield!
- Left click to dig. If there's any available space, you'll automatically dig up any safe squares that are available. 

## Current Goals
- Add distances to bombs, and store them in the matrix
- Highlight 2-4 squares that are guaranteed to be safe to click on. 
- Hide the squares
- Find out how Google and other programs figure out how vacant space is calculated.
